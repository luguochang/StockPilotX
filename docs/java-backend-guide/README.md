# Java 后端视角阅读指南

本目录用于帮助 Java 后端开发快速理解 `StockPilotX` 的 Python 实现。

## 建议阅读顺序
1. `01-项目结构与分层映射.md`
2. `02-请求链路与执行时序.md`
3. `03-hook与中间件机制.md`
4. `04-python语法对照java.md`
5. `05-调试与排查手册.md`
6. `06-新同学全景理解与数据流图.md`

## 适用对象
- 熟悉 Spring Boot / Java 分层架构
- 初次接触 Python、FastAPI、Agent 工作流

## 你会得到什么
- Java 概念和 Python 概念的一一对应
- 一条请求在系统中的完整流转路径
- hook/middleware 的执行模型（洋葱模型）
- 常见 Python 语法在本项目中的具体用法
